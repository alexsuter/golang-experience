## Installation
Um die Beispiele auszuführen, kann einfach der Go-Online-Compiler verwendet werden: <https://play.golang.org/>

Falls die Go-Programme direkt auf der Machine ausgeführt werden sollen, kann Go hier heruntergeladen werden: <https://golang.org/dl/>. Wählen Sie die betreffende Zielplattform und Go in der Version >1.5.
